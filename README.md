# 故易GoodsEasy------校园闲置物品交易商城

​	

## 商城介绍

故易是一个基于c2c模式的校园交易平台。故易，顾名思义，在这里，无需注册店铺就能卖闲置：为闲置的书本寻找新读者，把不再喜爱

的电脑配件转给玩机好友……在这里，你可以自主手机拍照上传二手闲置物品、以及在线交易等诸多功能。让闲置的宝贝以最快的速度奔

赴天南海北的新主人手中物尽其用。
